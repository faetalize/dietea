## App overview

A single-page vanilla JavaScript webapp for meal prep planning with optional ingredient syncing via the File System Access API.

## Getting started

1. Open `index.html` in a modern browser (Chrome/Edge recommended for File System API support)
2. Complete onboarding
3. (Optional) Connect `ingredients.json` from Settings for auto-sync

## Development

```bash
npm run dev
```
